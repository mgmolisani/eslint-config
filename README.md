# `@mgmolisani-private/prettier-config`

> My personal [ESLint](https://eslint.org) config.

## Usage

**Install**:

```bash
$ npm i -D @mgmolisani-private/eslint-config
```

**Edit `.eslintrc.js`**:

```javascript
module.exports = {
  // ...
  "extends": ["@mgmolisani-private/eslint-config"]
}
```
